# College Utility System 2026

A comprehensive college management system built with HTML, CSS, JavaScript, Java Spring Boot, and MySQL.

## Features

- **Authentication System**: JWT-based secure login for students
- **Student Dashboard**: Profile, attendance, grades, notices, and timetable
- **Admin Panel**: Manage students, attendance, grades, and notices
- **Responsive Design**: Green and white theme with mobile support

## Technology Stack

- **Frontend**: HTML5, CSS3, JavaScript
- **Backend**: Java Spring Boot
- **Database**: MySQL
- **Security**: JWT Authentication, Password Encryption

```
├── frontend/

└── database/
```
